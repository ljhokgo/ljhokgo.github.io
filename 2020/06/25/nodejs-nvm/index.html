nodejs 버전 관리도구 nvm

nodejs로 개발을 하다보면 다양한 node 또는 npm 버전을 사용해야 할 때가 있습니다.
사실 저 같은 경우 이럴 때 마다 필요한 버전의 node나 npm을 새로 설치했는데..
얼마전 알게된 nvm이라는 툴로 너무나 간단하게 이 문제가 해결되어 nvm 사용법을 간단하게나마 공유합니다.

nvm은 Node Version Manager의 약자로 이름에서 알 수 있다시피 nodejs의 버전을 관리해주는 툴입니다.
계정(user) 별로 설치되며, shell 별로 버전을 설정할 수 있고 간단한 명령어로 버전을 변경할 수 있습니다.
